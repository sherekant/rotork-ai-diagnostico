# Rotork AI - Diagnóstico Inteligente

Aplicación web basada en inteligencia artificial generativa para el diagnóstico de fallas en actuadores industriales Rotork (IQ3, CK y sistemas Pakscan).

## 🚀 Funcionalidades
- Diagnóstico de fallas mediante entrada de texto
- Análisis de imágenes de displays industriales
- Generación de respuestas estructuradas (diagnóstico, causas y recomendaciones)
- Traducción automática de términos técnicos
- Integración con información técnica del fabricante

## 🛠 Tecnologías utilizadas
- Python / Flask
- Groq API
- Modelos LLaMA (texto y visión)
- HTML / CSS / JavaScript
- BeautifulSoup (web scraping)
